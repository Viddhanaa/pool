Viddahana JSON‑RPC API

Chain ID: 202401 • Public endpoint: https://rpc.viddhana.com

Quick Start

Use HTTPS POST with Content‑Type: application/json. Follows Ethereum JSON‑RPC (EIP‑1474).

curl -s -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","method":"net_version","params":[],"id":1}' \
  https://rpc.viddhana.com

Common methods:

  • net_version → chain id
  • eth_blockNumber → latest block
  • eth_getBalance [address, block]
  • eth_getTransactionByHash [txHash]
  • eth_call to read contract state
